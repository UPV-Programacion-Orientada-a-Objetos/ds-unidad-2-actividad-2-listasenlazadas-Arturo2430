Descripción detallada

Clase base abstracta para tramas del protocolo PRT-7.

Esta clase define la interfaz común para todas las tramas que pueden ser recibidas a través del puerto serial. Cada tipo de trama debe implementar el método procesar() de acuerdo a su lógica específica.
