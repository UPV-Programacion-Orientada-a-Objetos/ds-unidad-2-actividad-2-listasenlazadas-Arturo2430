Descripción detallada

Aplicación principal - Decodificador PRT-7 (RECEPTOR)

Este programa recibe tramas del protocolo PRT-7 desde el puerto serial, las procesa usando el rotor de mapeo y decodifica el mensaje original.

Formato de tramas:

  • L,<caracter> : Carga un carácter (ej: L,H o L,Space)
  • M,<numero> : Rota el rotor (ej: M,2 o M,-2)
