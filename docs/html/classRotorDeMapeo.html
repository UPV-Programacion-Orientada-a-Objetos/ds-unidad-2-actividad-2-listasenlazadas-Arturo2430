Descripción detallada

Lista circular para mapeo de caracteres (Rueda de César)

Implementa un rotor de cifrado que contiene el alfabeto A-Z más el espacio. Puede rotar su posición para cambiar el mapeo de caracteres, implementando un cifrado César dinámico.
